# StudySync

## Description

StudySync is a smart study planner that helps students organize daily study tasks, create study schedules, manage revision plans, and track learning progress. It integrates productivity tools like a Pomodoro timer, study streak tracking, reminders, and adaptive study planning to improve consistency and learning efficiency.

## Features

- User authentication (Sign Up/Login)
- Personalized dashboard
- Daily task management
- Study schedule planner
- Smart adaptive task rescheduling
- Smart revision planner using spaced repetition
- Pomodoro timer
- Study streak tracking
- Progress tracking and analytics
- Study reminders and overdue task notifications
- Responsive design for desktop and mobile

## Technologies Used

## Algorithms & Concepts Used

- Spaced Repetition Algorithm

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Navigate to the backend folder.
4. Copy `.env.example` to `.env`.
5. Configure the MongoDB connection string and JWT secret in the `.env` file.
6. Install backend dependencies: npm install
7. 7. Start the backend server: npm run dev
8. Open a new terminal and navigate to the frontend folder.
9. Install frontend dependencies: npm install
10. Start the frontend: npm run dev
11. Open the application in your browser using the local URL displayed by Vite.
